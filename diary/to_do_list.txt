+ календарь, при нажатии на дату которого появляется список задач в правом table box(table view)
- база данных задач(sqlite)
    --task
        ---id_task (int, autoincrement, not null) номер задачи по порядку
        ---name_task (text) имя задачи кратко
        ---name_description(text) описание задачи
        ---rank_task(int) важность задачи, чем больше число, тем более важна задача)
        ---check_task(int) отметка о выполнении красная - не выполнена, желтая - в процессе выполнения,
                            зеленая - выполнена
        ---start_date(datetime) дата начала выполнения задачи
        ---end_date(datetime) дата окончанния выполнения задачи
    --human
        ---id_human (int, autoincrement, not null) номер человека по порядку
        ---name_human (str) имя человека
        ---surname_human (str) фамилия человека
        ---fathers_name_human (str) отчество человека
        ---human_description (text) описание человека
        ---birthday_date (datetime) день рождения человека
        ---telephone_num (str) контактный телефон
        ---human_group (text) группа человека

- добавление задачи
- удаление задачи
- изменение описания задачи
- постановка отметки о выполнении
- поиск задачи глабально(нижнее поле) по параметрам(название)
- поиск задачи локально для правого окна, среди найденных задач
- после поиска и отображения списка задач в нижнем окне, на задачу можно кликнуть и после двойного
    нажатия календарь переходит на дату задачи
- сортировка задач по категориям
- id_task не видна(добавить просто номерацию задач)
- возврат к текущей дате
- добавление формы дней рождений(переход через кнопку основного окна)
- radio botton выбора режима работа (Список задач/Дни Рождения)
- для дней рождений по нажатию на дату выбираются 9 следующих дней
